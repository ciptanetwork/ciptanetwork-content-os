# SEO On-Page Standards

## Struktur URL (Slug)
- Gunakan bahasa sesuai site (ID untuk site Indonesia, EN untuk site Inggris).
- Format: `domain.com/kategori/primary-keyword`
- Panjang maksimal 5 kata.
- Contoh EN: `ciptavisual.com/design/brand-identity-tips`
- Contoh ID: `kitiranmedia.com/ai/prompt-midjourney-terbaik`

## Meta Data
- **Title Tag:** 50-60 karakter. Harus mengandung Primary Keyword di awal.
- **Meta Description:** 150-160 karakter. Mengandung Primary + Secondary Keyword, dan ajakan bertindak (CTA) yang jelas.
- **Focus Keyphrase:** 1 keyword utama per artikel.

## Struktur Heading
- **H1:** Hanya 1. Sama dengan judul artikel.
- **H2:** Sub-topik utama (minimal 3-4 H2 per artikel).
- **H3:** Sub-bagian dari H2 (untuk detail lebih dalam).

## Internal Linking
- Setiap artikel WAJIB memiliki minimal 2 link ke artikel lain di website yang sama (internal link).
- Minimal 1 link ke halaman utama (Homepage) atau halaman pillar (cornerstone).
- Gunakan anchor text yang natural (bukan "klik di sini").

## Gambar (Image SEO)
- Nama file gambar wajib deskriptif (contoh: `brand-identity-tips.jpg`).
- Wajib mengisi **Alt Text** yang mengandung keyword relevan.
- Ukuran file dioptimalkan (< 200KB) agar loading cepat.
