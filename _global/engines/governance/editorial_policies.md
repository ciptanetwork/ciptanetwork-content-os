# Global Editorial Policies

## Cross-Site Standards

### Language Quality
- All content must be NATURAL in the target language
- NO machine translation from another language
- Write directly in the target language
- Use native-level fluency and cultural relevance

### Quality Assurance
- All articles must pass human editing checklist
- AI-generated content must be "de-AI-ified" (see de_ai_ification_techniques.md)
- Fact-check all claims and statistics
- Verify all tool/software recommendations

### Content Uniqueness
- 4 sites CAN share topics but with different angles
- NO duplicate content across sites
- Each site must have unique perspective, examples, and voice
- Cross-link appropriately when relevant

### Brand Safety
- No offensive or harmful content
- Respect cultural sensitivities per market
- Cite sources properly
- Disclose AI usage if applicable

## Site-Specific Overrides
See each site's `_editorial_policies.md` for specific tone, language, and policy exceptions.

## Version Control
- All content drafts stored in GitHub
- Use feature branches for new articles
- Main branch = published content only
- Archive drafts after 90 days if unpublished
