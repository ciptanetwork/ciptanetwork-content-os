# Versioning Rules

## Nama File
Format: `YYYY-MM-DD-slug-artikel_v[Angka].md`
Contoh: `2025-05-20-brand-identity-tips_v1.md`

## Aturan Versioning
- **v1:** Draft pertama dari AI.
- **v2:** Setelah diedit oleh Human Editor (jika ada).
- **v3:** Setelah revisi berdasarkan data terbaru (misal: update tools).

## Changelog
Setiap kali update, tulis changelog di bagian bawah file:

```markdown
## Changelog
- **v1 (20-05-2025):** Initial draft.
- **v2 (22-05-2025):** Added new section about Canva AI.
