# WordPress Publish Steps

## 1. Login
- Masuk ke `/wp-admin` masing-masing site.

## 2. Add New Post
- Klik "Add New Post".

## 3. Isi Konten
- Paste konten dari Markdown ke Editor (Gutenberg/Classic).
- Pastikan formatting (bold, italic, heading) tidak hilang.

## 4. Setel Featured Image
- Klik "Set Featured Image".
- Upload gambar sesuai standar visual (1200x628).

## 5. Atur SEO (Jika pakai Yoast/Rank Math)
- Paste Meta Title dan Meta Description.
- Set Focus Keyphrase.
- Pastikan traffic light hijau (minimal oranye).

## 6. Kategori & Tag
- Pilih kategori utama.
- Tambahkan tag yang relevan.

## 7. Publikasi
- Jika masih draft: Klik "Save Draft".
- Jika sudah final: Atur tanggal schedule (misal: besok pagi jam 8) atau klik "Publish" langsung.

## 8. Cek di Front-End
- Buka artikel di incognito mode untuk memastikan tidak ada error tampilan.
